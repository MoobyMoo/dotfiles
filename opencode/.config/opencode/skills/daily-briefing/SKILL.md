## Purpose

Create a useful morning or catch-up briefing that reduces context switching.

## Inputs

- Calendar events for today and tomorrow
- Important unread or recent emails
- Active commitments from operational memory
- Relevant notes or files from Drive or Obsidian when available

## Output sections

- Today at a glance
- Scheduling risks and conflicts
- Important emails needing action
- Follow-ups to draft
- Top 3 priorities
- Memory candidates, if any

## Execution notes

- Keep it short enough to scan in under two minutes.
- Highlight decisions, not just information.
- When tool access is missing, ask for or use available local sources instead.

## Safety

- Read and draft only by default.
- Do not send messages or modify calendar entries without approval.
